Một phần mềm Source-Available phục vụ cho LiveStream và Donate

# 📦 Cài đặt

Để phát triển phần mềm này cần:
| Runtime  | Supported |
| -------- | --------- |
| Nodejs   | ✓         |
| Bun      | ✓         |
| Python   | ✓         |
| Rust     | ✓         |

# 🔧 Hướng dẫn Xây dựng

Để xây dựng hãy chạy:

```bash
bun install
bun tauri dev
```

Còn đóng gói hãy chạy:

```bash
bun tauri build
```

# 🧩 Kiến trúc hệ thống

- **Frontend**: Tanstack
- **Backend**: Rust
- **Sidecar**: nguyenvulebinh/VietVoice-TTS + Cloudflared Tunnel + NodeJS
